# ASTRA

Local-first AI assistant created by Ashwanth.

## Features

* Offline speech recognition using Faster-Whisper
* Local AI processing using Qwen 3 4B via Ollama
* Wake word activation ("Astra")
* Natural voice responses using Edge-TTS
* VLC audio playback
* Sleep mode
* Exit command
* Fully local execution

## Tech Stack

* Python
* Faster-Whisper
* Ollama
* Qwen 3 4B
* Edge-TTS
* VLC
* SoundDevice
