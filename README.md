# ASTRA

A local-first AI assistant created by Ashwanth.

## Features

### Voice Interaction

* Wake word activation ("Astra")
* Offline speech recognition using Faster-Whisper
* Natural voice responses using Edge-TTS
* VLC-based audio playback

### AI Capabilities

* Local AI processing using Qwen 3 4B via Ollama
* General question answering
* Coding assistance
* Educational explanations
* Conversational interaction

### PC Automation

* Open installed applications
* Discover installed applications automatically
* Open websites
* Internet search fallback for unknown targets
* Play songs on YouTube via voice commands

### Assistant Controls

* Sleep mode
* Exit command
* Fully local execution

## Tech Stack

* Python
* Faster-Whisper
* Ollama
* Qwen 3 4B
* Edge-TTS
* VLC
* SoundDevice
